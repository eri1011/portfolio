出路針茶房(架空のカフェサイト)

■URL

出路針茶房（※別窓で開きます）

■担当

ワイヤーフレーム、デザインと原稿の作成、コーディング

■使用ツール

Photoshop (ワイヤーフレーム、デザインカンプ、写真の加工、書き出し) 、 Visual Studio Code (コーディング)

■使用コード

HTML (マークアップ) 、 CSS (スタイルの指定、動作の付与、Sass記法でコーディング) 、 JavaScript (動作の付与)

■制作期間

ワイヤーフレームとデザイン(5日) 、 コーディング(4日)

■目的

和風カフェサイトの店舗の宣伝、集客

■ターゲット

抹茶、ほうじ茶といった日本茶や、それらを使用したスイーツが好きな人。
街の中心部で本格的な日本茶を楽しみたい人。

■制作物について

和風サイトのデザインの練習と、JavaScriptを使用したモーダルの学習のため、和風カフェサイトを自主制作いたしました。

「お知らせ」の各リンクを押下すると、お知らせの詳細文がモーダルとして表示されるという処理を付与しました。モーダルの作成にあたり、「Micromodal.js」というjQueryを使用しないモーダルライブラリを使用しました。
CSSは、公式GitHubページのCSSサンプルコードを引用し、サイトのデザインに合わせてカスタマイズしています。

デザイン面で工夫した点は、ページ内で使用している写真の表示方法です。
トップ画面はヘッダーを表示させずに、抹茶の写真を一面に出したいと考えました。そのため、一定量下部にスクロールしてからヘッダーを表示させる処理をCSSとJavaScriptで付与しました。
